# HONCA_reproducibility

This repository comprises code and analyses for the Human Optic Nerve Cell Atlas (HONCA) project within the HCA framework, emphasizing the [Eye Biological Network](https://www.humancellatlas.org/biological-networks/).

## File organization for figures and tables

The main code can be found in the [analysis](./analysis) directory of this repository. The subdirectories are organized according to the main figures corresponding to the manuscript. The [utility](./utility) folder comprises standalone scripts that have been used in multiple places.

## HONCA on Interactive Browsers

HONCA can be accessed through [CELLxGENE](https://cellxgene.cziscience.com/collections/05e3d0fc-c9dd-4f14-9163-2b242b3bb5c2) and [CAP](https://celltype.info/project/621).
